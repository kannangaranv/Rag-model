# FastAPI Project Setup

## 1. Create Virtual Environment (Command Prompt)
```cmd
python -m venv .venv
```
 ## 2. Activate Virtual Environment

```bash
.venv\Scripts\activate.bat
```

## 3. Install packages 
```bash
pip install -r requirements.txt
```

## 4. Run the project
```bash
uvicorn app.main:app --reload --port 8000
```
